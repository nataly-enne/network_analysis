## Build setup (testar localmente):
```bash
# python virtual environment
python3 -m venv env

# activate the environment
source env/bin/activate

# install dependencies
pip3 install -r requirements.txt

# run app
streamlit run streamlit_analysis.py

# stop environment
deactivate
```
